# Contact Management System

A full-stack Contact Management System developed as part of my internship at Maincrafts Technology. This project allows users to submit contact details through a web form, stores the data in a MySQL database, and provides an admin dashboard to manage submissions.

---

## 🚀 Features

- Contact Form Submission
- Client-Side Validation
- Server-Side Validation
- Spring Boot REST API
- MySQL Database Integration
- Spring Data JPA & Hibernate
- Admin Dashboard
- Search Contacts
- Delete Contacts
- Submission Timestamp Tracking
- Swagger/OpenAPI Documentation
- Responsive User Interface
